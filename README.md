## 简单的备忘录
    技术实现：react+react-router+redux+postcss+es6
    在线演示地址：[备忘录](http://wsw.github.io/memorandum/index.html)

###功能说明
    支持回车添加新事项;
    支持删除事项(点击X符号);
    支持状态转换具体包括：
    正在进行->已完成(点击文字内容本身)
    已完成->正在进行(点击文字本身)
    支持判断输入空字符
    支持本地化存储;
    支持左侧菜单以及路由切换；

###本地演示方法
    # 安装依赖
    npm install
    # 本地运行
    npm run start
    # 运行地址
    localhost:8080
